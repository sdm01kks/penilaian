Input Nilai SLM & SAS

Input nilai setiap Tujuan Pembelajaran saat SLM dan SAS. SAS bersifat opsional — jika tidak diisi, nilai SLM menjadi nilai akhir TP tersebut.

—

📊 Import / Export Nilai

Cara kerja: Export template → guru isi nilai di Excel → import kembali. Sistem akan memvalidasi setiap nilai sebelum disimpan.
📊

Seret file Excel (.xlsx) ke sini atau

—
📌 Cara kerja nilai: Setiap TP memiliki nilai SLM (wajib) dan nilai SAS (opsional). Jika hanya ada SLM → nilai akhir TP = SLM. Jika ada keduanya → nilai akhir TP = (SLM × bobot%) + (SAS × bobot%). Nilai akhir mapel = rata-rata nilai akhir semua TP.
0 TP SLM+SAS lengkap
0 TP hanya SLM
0 TP belum ada nilai
